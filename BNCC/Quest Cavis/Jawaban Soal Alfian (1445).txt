Jawaban Soal 2:
1. API (Application Programming Interface) adalah penghubung antara pengguna/user ke sebuah server, perumpamaan nya seperti ini, kita datang ke sebuah restoran lalu memanggil waiter/pelayan, kita menyebutkan pesanan kita kepada waiter lalu waiter akan menyampaikan pesanan kita tersebut ke dapur, setelah makanan yang kita pesan sudah jadi, waiter tadi akan mengantarkan makanannya kepada kita. Nah, waiter dalam perumpamaan ini adalah API, dapur adalah sebuah server dan kita adalah user.
2. Client adalah komputer/perangkat yang digunakan oleh user untuk terhubung dan melakukan request suatu layanan ke server. Sederhananya client adalah pengguna jasa dari sebuah server. Server sendiri adalah sistem yang menyediakan layanan tertentu untuk dapat digunakan oleh client. Jadi, kesimpulannya client adalah yang meminta sesuatu dan server adalah yang menyediakan sesuatu tersebut.
3. Dedicated server, VPS dan shared hosting adalah produk komputasi server. Ketiga ini memiliki perbedaan yang cukup mudah dimengerti, dedicated server adalah sebuah server dengan resource yang besar dan menggunakan server fisik. VPS atau Virtual Private Server adalah server dengan resource yang lebih kecil dibandingkan dedicated server karena VPS adalah hasil dari dedicated server yang dibagi, di VPS ini kita memegang kontrol penuh atas server kita. Sedangkan shared hosting adalah hasil dari VPS yang dibagi-bagi sehingga resourcenya pun terbagi dengan pengguna shared hosting lainnya dalam satu VPS.
